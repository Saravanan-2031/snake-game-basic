# Snake Game (Python + Pygame)

A classic Snake game built using Python and Pygame.

## Features
- Grid-based movement
- Food spawning
- Score tracking
- High score system
- Self-collision detection
- Restart option

## Controls
- Arrow keys → Move
- R → Restart
- Q → Quit

## How to Run
1. Install pygame:
   pip install pygame
2. Run:
   python main.py
